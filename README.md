# React + TypeScript + Vite

## Getting Started

### Install dependencies

```bash
pnpm install
```

### Start development website

```bash
pnpm run dev
```

### Build production website

```bash
pnpm run build
```

## Description

### Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint
- Prettier
- Axios
- TanStack/react-router
- React Spinners
- ShadCn ui
